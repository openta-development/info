Multiple choice questions contains different alternatives where one or more can be correct. They are answered by marking the correct alternatives and then submitting. The content of the alternatives can be text, math or elements such as figures.

<question type="multipleChoice">
  <text>Pick the correct alternatives below.</text>
  <choice key="0"><text> The first choice</text></choice>
  <choice key="1" correct="true"><text> The second choice</text></choice>
  <choice key="2"><text> The third choice</text></choice>
  <choice key="3" correct="true"><text> The fourth choice</text></choice>
</question>

Specifies how many tries a student can make per duration of time. The time is specified as number/unit where unit is s (second) or h (hour). For example <rate>3/h</rate> permits three tries per hour. See rates for the detailed syntax description.
